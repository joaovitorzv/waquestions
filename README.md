# WaQuestions


Waproject Front End technical test [see live](https://waquestions.netlify.app/).


## Run locally
Clone this repo
```bash
git clone git@github.com:joaovitorzv/waquestions.git
```

Download all dependencies
```bash
yarn add
```

Start the project
```bash
yarn start
```
